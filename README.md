# Rnn_fromscratch
# Mini RNN from Scratch pour la prédiction du prochain mot  Ce projet implémente un petit réseau de neurones récurrent (**RNN**) codé presque from scratch avec PyTorch pour faire de la **prédiction du prochain mot**.  ## Objectif  À partir d'une séquence de tokens, le modèle apprend à prédire le token suivant à chaque pas de temps.
